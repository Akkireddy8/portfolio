About

Senior Java Full Stack Developer

I am an AWS Certified Solutions Architect and Senior Java Full Stack Developer with 8+ years of experience designing and developing scalable, secure, and high-performance enterprise applications across industries including education, healthcare, finance, and government.

My technical expertise spans both frontend and backend development. I have built modern, responsive web applications using React.js, Angular, Redux, and RxJS, applying best practices in performance optimization, reusable UI component design, and accessibility.

On the backend, I specialize in building RESTful APIs and microservices using Java 8/11/17, Spring Boot, Spring Security, and Hibernate ORM. I implement robust security mechanisms such as OAuth2, JWT, and OpenID Connect to enforce role-based access and authentication across systems.

I bring hands-on experience with distributed architectures using Spring Cloud (Eureka, Config Server, Gateway, Feign) and integrate messaging systems like Apache Kafka, RabbitMQ, and ActiveMQ to support event-driven workflows.

I work with relational and NoSQL databases such as PostgreSQL, MySQL, Oracle, and MongoDB, and leverage caching tools like Redis for enhanced backend performance.

I’m experienced in deploying and maintaining services on cloud platforms including AWS and Azure, using tools like Jenkins, GitHub Actions, and Docker to automate CI/CD pipelines and ensure reliable delivery.

Currently, I am working at Cengage Learning, where I contribute to the InstructIQ Authoring Platform — a cutting-edge cloud-based solution enabling educators to create interactive and personalized learning experiences at scale.
